# docker-jupyter

Docker image of jupyter with scipy, numpy, pandas, matplotlib and IPAexGothic Japanese font.

---

# Build

```
$ docker-compose build
$ docker-compose up
```

# How to use

```
$ open `docker-machine ip default`:8888
```
